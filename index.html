What Can I Do

HTML / CSS

超级文本标记语言(HTML)是标准通用标记语言下的一个应用，也是一种规范，一种标准。
CSS 能够对网页中元素位置的排版进行像素级精确控制。

JS / jQuery

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。
jQuery是一个快速、简洁的JavaScript框架。

HTML5 / CSS3 / Bootstrap

HTML5与CSS3是HTML和CSS的升级
Bootstrap 是基于 HTML、CSS、JavaScript 的一个CSS/HTML框架。
它在jQuery的基础上进行了更为个性化的完善.

ajax

AJAX即“Asynchronous Javascript And XML”（异步JavaScript和XML），是指一种创建交互式网页应用的网页开发技术。

gulp

gulp是前端开发过程中对代码进行构建的工具，是自动化项目的构建利器；它不仅能对网站资源进行优化，而且在开发过程中很多重复的任务能够使用正确的工具自动完成。

angular

AngularJS 是一个 JavaScript 框架。它是一个以 JavaScript 编写的库。
AngularJS有着诸多特性，最为核心的是：MVC、模块化、自动化双向数据绑定、语义化标签、依赖注入等等。
